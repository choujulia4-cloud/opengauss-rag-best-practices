# opengauss-rag-best-practices
基于 openGauss 的 RAG / RetrievalQA 工程实验与最佳实践，涵盖文档切分、向量检索、生成式问答、服务封装与轻量化部署探索。Engineering experiments and best practices for an openGauss-based RAG / RetrievalQA pipeline, covering document chunking, vector retrieval, answer generation, service packaging, and lightweight deployment.
