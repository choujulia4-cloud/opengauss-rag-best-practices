# opengauss-rag-best-practices
中文：
本仓库用于整理和实现一个基于 openGauss 的 RAG / RetrievalQA 项目最佳实践。项目围绕文档切分、embedding 生成、向量检索、生成式问答、BentoML 服务封装以及 AArch64 / 边缘设备部署等环节展开，重点关注不同配置下的系统效果、工程可复现性与部署可行性。除代码实现外，仓库还将持续沉淀实验记录、评测结果、配置对比与实践总结，用于支持后续优化、项目复现与社区分享。

English:
This repository focuses on the implementation and best-practice exploration of an openGauss-based RAG / RetrievalQA system. The project covers document chunking, embedding generation, vector retrieval, answer generation, BentoML-based service packaging, and deployment on AArch64 / edge devices. It emphasizes system effectiveness under different configurations, engineering reproducibility, and deployment feasibility. In addition to source code, the repository will continuously collect experiment records, evaluation results, configuration comparisons, and practical conclusions to support future optimization, reproduction, and community sharing.
